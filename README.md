# luvas.io

## APP

Project's frontend. Created with Angular.

#### Setup

> Check luvas.io-app/README.md for more information.

> Check luvas.io-app/src/glass/README.md for more information.

```bash
cd luvas.io-app
npm install
```

## API

Project's backend. Created with Node.js and Express.

#### Setup

> Check luvas.io-api/README.md for more information.

```bash
cd luvas.io-api
npm install
```

## GENERAL

#### Run DEV

Open two terminals!

```bash
cd luvas.io-api
npm run dev
```

```bash
cd luvas.io-app
ng serve
```

#### Run PROD

Only one terminal!

```bash
cd luvas.io-app
ng build
```

```bash
cd ..
cd luvas.io-api
npm run prod
```
